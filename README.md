# MESS-MEAL-PLANNING-AND-FOOD-WASTE-REDUCTION-SYSTEM-USING-JAVA
Java GUI application to estimate mess meal demand and reduce food waste using demand prediction factors.
This project is a Java GUI application that predicts mess meal demand based on factors like special meal days, academic periods, and meal timing.
Features:
- Predict expected meals
- Calculate food waste
- Waste percentage analysis
- Donation suggestion
